## Objective
Build a machine learning model to predict customer churn using
demographic, service usage, and billing information.

## Dataset
IBM Telco Customer Churn dataset.

The dataset is not included in this repository due to GitHub file size limits.
Download from:
https://www.kaggle.com/blastchar/telco-customer-churn

Place the CSV file in:
Project/Churn_Prediction/data/

## Models Used
- Logistic Regression
- Random Forest Classifier

## Techniques
- Label Encoding
- Feature Scaling
- SMOTE for class imbalance
- Model evaluation using accuracy, precision, recall, F1-score

## Tools
Python, Pandas, NumPy, Scikit-learn, Streamlit
